# @grupo-ser/types

Shared TypeScript type definitions for Grupo Ser monorepo.

## Usage

```typescript
import type {
  CourseData,
  CoursesResponse,
  CourseDetailsResponse,
  CourseModality,
  AutocompleteResponse,
} from "@grupo-ser/types";
```

## Installation

This package should be installed as a **runtime dependency**:

```bash
yarn add @grupo-ser/types
```

## Development

```bash
# Build
yarn build

# Type check
yarn typecheck

# Clean
yarn clean
```
